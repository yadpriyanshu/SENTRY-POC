
# Sentry-POC

Sentry is a cross-platform crash reporting and aggregation platform.
This repository aims to host Sentry Application on GKE.

To deploy Sentry Application on GKE using Redis and Postgres.
1. Redis (MemCached)
2. Postgres (CloudSQL)
3. Sentry
